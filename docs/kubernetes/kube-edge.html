主要 修改容器运行时、cgroup 驱动

运行 

edgecore

如下错误：

edgecore[45472]: E0407 06:52:35.569727 45472 edged.go:742] Failed to start container manager, err: system validation failed - Following Cgroup subsystem not mounted: [memory] edgecore[45472]: E0407 06:52:35.569777 45472 edged.go:293] initialize module error: system validation failed - Following Cgroup subsystem not mounted: [memory]

# 内存资源的cgroups机制没有开启

# 修改启动参数
$ vim /boot/cmdline.txt

# 追加
cgroup_memory=1 cgroup_enable=memory


# 查看可用的cgroup
$ cat /proc/cgroups
scp k8s-master:/etc/cni/net.d/* /etc/cni/net.d/
